## Session #1 — 2026-03-27

**Mục tiêu**: Tìm kiếm các tính năng mới nhất của n8n trong quý 1/2026 và các cập nhật lớn từ cuối 2025 (n8n 2.0).
**Phương pháp**: Web search, phân tích release notes và các bài blog công nghệ.
**Nguồn chính**: n8n.io, GitHub, các diễn đàn automation và AI.
**Phát hiện chính**:
- n8n 2.0 chính thức ra mắt (Stable đầu 2026) với thay đổi lớn về kiến trúc: Task Runners (cô lập thực thi), SQLite pooling (tăng tốc DB 10x).
- AI-First Strategy: Hơn 70 node AI, tích hợp LangChain bản địa, AI Workflow Builder (tạo workflow bằng ngôn ngữ tự nhiên).
- Tính năng Autosave & Publish/Save paradigm: Tách biệt môi trường Draft và Production.
- MCP (Model Context Protocol) support: Cho phép các AI agent bên ngoài kích hoạt workflow n8n.
**Đánh giá**: Dữ liệu rất phong phú và có độ tin cậy cao, n8n đã chuyển mình mạnh mẽ thành một nền tảng AI Orchestration.
